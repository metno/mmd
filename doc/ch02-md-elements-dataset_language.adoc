[[dataset_language]]
=== dataset_language

[cols="2,8"]
|=======================================================================
|Element |/mmd/dataset_language

|Attributes |None

|Required |No

|Repetition allowed |No

|Description |The ISO 639-1 two letter language code of the language used in production, storage etc. of the
dataset. The default for all datasets is English (en).

|Example XML: a|
----
<dataset_language>en</dataset_language>
----

|DIF equivalent |/DIF/Data_Set_Language

|ISO equivalent |/gmd:MD_Metadata
